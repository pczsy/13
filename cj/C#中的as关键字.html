    你能进到这篇文章……

    首先你可能明白类似于

    double num=0;
    int m=(int) num;
     

    或者你明白 

    int m=209;
    String str=m.toString();
    或者说你甚至见过

    char m='c';
    int m=Integer.parseInt(m);
    现在应该明白as了吧？ 对， 你没有猜错，它就是一种类型强转的方式。例子如下：

    Object object=null;
    string s = object as string;
    if (s != null)
    {
    // object is a string.
    }
