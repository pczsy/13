    • C#控件——批量化隐藏或显示同类型控件
    原文出处:本文由博客园博主虎虎虎提供。
    原文连接:https://www.cnblogs.com/dobiprogrammer/p/10768222.html

    当一个页面中添加了许多同类型控件，当需要控制这些控件进行显示或隐藏的时候，需要一个个的将Visible属性设置为false,十分不方便，

    后通过论坛受一位大神（至于叫什么忘了）的启发，通过建立控件数组的方式可以实现同种控件批量修改同一属性。

    ToolStripStatusLabel[] Tlable = new ToolStripStatusLabel[]{lable_1,lable_2,lable_3,lable_4,lable_5,lable_6,lable_7,lable_8};
    for (int i = 0; i <Tlable.Length; i++)
    {
    Tlable[i].Visible = false;
    }

    可以实现将数组中所有的ToolStripStatusLabel控件设置为隐藏。
