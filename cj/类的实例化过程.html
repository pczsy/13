    • 容易遇到的问题

    1. 当父类没有无参构造函数，继承父类的子类必须要调用父类的构造函数，否则会报错
        public class Parent
    {
        public Parent(string a)
        {
            Console.WriteLine(a);
        }
    }
    public class Son : Parent
    {
        public Son(string a) : base(a)
        {
            Console.WriteLine(a);
        }
    }
