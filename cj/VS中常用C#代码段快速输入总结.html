    Visual Studio 中有很多代码段都可以直接简写然后按TAB快速输入编译器中，为了提高编程效率，特此查阅资料，罗列总结。

     10.   foreach             创建foreach循环

            foreach (var itemin collection)

                    {               

                    }

    var        代表要循环访问的集合中对象的类型

    collection   要循环访问的集合或数组的名称
