    搭建环境：Windows 10

    参与文档：https://github.com/exceptionless/Exceptionless/wiki/Self-Hosting

    运行环境：

    1. .NET 4.6.1
    2. Java JDK 1.8(如果用此文章中的exceptionless，JDK版本必须为1.8)
    3. IIS7.5+
    4. ElasticSearch

    下载地址：链接: https://pan.baidu.com/s/16-OM9sIj-YxyQ3-hbfPhhw 提取码: mgvj

    下载压缩包后解压，运行Start.bat，顺利的话会直接启动Exceptionless

     

    如果想部署到IIS，则需要要IIS添加一个站点指向wwwroot目录，并修改wwwroot下app.config.*.js中的BASE_URL为IIS站点访问URL

    关闭防火墙

     

    elasticsearch想安装成 windows服务的话，到elasticsearch-5.3.0-node1/bin下运行elasticsearch-service.bat install

     

    问题：运行Start.bat时，The request was aborted: Could not create SSL/TLS secure channel

    解决：在Start-ElasticSearch.ps1顶部Param后面加[Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12

    问题：内存不够用
    解决：elasticsearch-5.3.0-node1\config\jvm.options修改-Xms200m -Xmx500m
