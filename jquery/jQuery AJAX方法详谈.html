    AJAX是与服务器交换数据并更新部分网页的技术，而无需重新加载整个页面。

    下表列出了所有jQuery AJAX方法：
    方法描述
    $.ajax() 执行异步AJAX请求
    $.ajaxPrefilter() 在发送每个请求之前以及由$.ajax()处理之前，处理自定义Ajax选项或修改现有选项。
    $.ajaxSetup() 为将来的AJAX请求设置默认值
    $.ajaxTransport() 创建一个处理Ajax数据实际传输的对象
    $.get() 使用AJAX HTTP GET请求从服务器加载数据
    $.getJSON() 使用HTTP GET请求从服务器加载JSON编码的数据
    $.parseJSON() 在版本3.0中已弃用，请改用JSON.parse()。 接受格式正确的JSON字符串并返回生成的JavaScript值
    $.getScript() 使用AJAX HTTP GET请求从服务器加载（并执行）JavaScript
    $.param() 创建数组或对象的序列化表示形式（可用作AJAX请求的URL查询字符串）
    $.post() 使用AJAX HTTP POST请求从服务器加载数据
    ajaxComplete() 指定AJAX请求完成时要运行的函数
    ajaxError() 指定当AJAX请求完成并出现错误时要运行的函数
    ajaxSend() 指定要在发送AJAX请求之前运行的函数
    ajaxStart() 指定第一个AJAX请求开始时要运行的函数
    ajaxStop() 指定所有AJAX请求完成后要运行的函数
    ajaxSuccess() 指定当AJAX请求成功完成时要运行的函数
    load() 从服务器加载数据，并将返回的数据放入所选元素
    serialize() 将一组表单元素编码为字符串以便提交
    serializeArray() 将一组表单元素编码为名称和值的数组
