    【问题】jQuery 名称发生冲突怎么办？

    【答案】jQuery 使用 $ 符号作为 jQuery 的简介方式。某些其他 JavaScript 库中的函数（比如 Prototype）同样使用 $ 符号。jQuery 使用名为 noConflict() 的方法来解决该问题。

                  var jq=jQuery.noConflict()，帮助您使用自己的名称（比如 jq）来代替 $ 符号。
