    • 弹出框 点击其他部分关闭
    原文出处:本文由博客园博主匍匐前进的小技术提供。
    原文连接:https://www.cnblogs.com/clt-web/p/11243702.html

    做网页的时候常常会遇到半透明背景色的弹出框，通常回会想要点击关闭按钮或者背景色部分关闭弹窗  并且弹窗是背景色的后代

    思路： 背景色盒子设置一个class类，通过jq 的点击事件 获取事件源对象(就是当前点击的) 判断点击的是否是 背景色盒子 这里通过类名判断

    $("背景色盒子的类").click(function(e) {
    　　if ($(e.target).attr('class') == '类名') {
    　　　　$(".背景盒子").hide();
    　　}
    })
