    • append() - 在被选元素的结尾插入内容
    • prepend() - 在被选元素的开头插入内容
    • after() - 在被选元素之后插入内容
    • before() - 在被选元素之前插入内容
    • first(), last() 和 eq()寻找第一最后一个及任意位置eq(n);
    • filter() 和 not() 选取匹配或不匹配的元素
    • remove() - 删除被选元素（及其子元素）
    • empty() - 从被选元素中删除子元素
    • hide()和show()显示隐藏 toggle()可以切换hide|show
    • fadeIn()淡入
    • fadeOut()淡出
    • fadeToggle()同toggle()
    • fadeTo()//可以设置透明度 $("#div3").fadeTo("slow",0.7)
    • slideDown()向下滑动
    • slideUp()向上滑动
    • slideToggle()同toggle()
    • animate() 创建动画
    • stop() 适用于所有 jQuery 效果函数，包括滑动、淡入淡出和自定义动画
    • text()  文本内容
    • html()  内容
    • val() 值
    • addClass() 添加一个或多个类
    • removeClass() 删除一个或多个类
    • toggleClass() 同toggle()
    • css() 设置或返回样式属性
    • width()设置或返回元素的宽度（不包括内边距、边框或外边距）
    • height()设置或返回元素的高度（不包括内边距、边框或外边距）
    • innerWidth()返回元素的宽度（包括内边距）
    • innerHeight()返回元素的高度（包括内边距）
    • outerWidth()返回元素的宽度（包括内边距、边框和外边距）
    • outerHeight()返回元素的高度（包括内边距、边框和外边距）
    • parent()返回父元素
    • parents()返回所有父元素
    • parentsUntil() 返回<span> 与 <div> 元素之间的所有父元素
    • children()和find()返回子元素
    • siblings()返回标签的所有同胞元素
    • next()返回下一个同胞元素
    • nextAll()例 返回所有跟随p标签后的元素
    • nextUntil()例 <h2> 与 <h6> 元素之间的所有同胞元素
    • prev()与prev()和prevUntil()上3个的方法类似,方向相反
                filter() 和 not()  demo

    　　    $("div").filter("p").css{"color",red};//匹配div下存在p元素的所有标签字体颜色为red

    　　    $("div").not("p").css{"color",red};//取反 不是p标签的为red
